Projetos Aqui você encontrará alguns dos projetos pessoais.

Social Dev

Um projeto pessoal criado em React e Node, com o intuito de expandir tanto meu conhecimento como experiência em projetos reais e de grande porte. Neste sistema eu utilizei o React no Frontend, fazendo a componentização e estilizando-as com ferramentas que são disponibilizadas pela biblioteca. O Backend eu utilizei Node fazendo requizições principalmente integrando o banco de dados com o frontend. Ao todo pude usurfruir de várias ferramentas como useState, Express, useEffcet, API_REST, StyledComponents, entre outras. Banco de Dados utilizado foi o MongoDB e hospedado pela Vercel.

Jogo da Velha

Um mini jogo da velha criado com react e um pouco de node, usando ferramentas, estados e efeitos que são bibliotecas oferecidas pelo React. Foi um grande desafio, pois estava aprendendo a biblioteca e achei importante mesmo com um pouco de conhecimento ja começar um projeto desse nível. Pude aprender muito sobre Express, requisições versionamento e principalmente deixar o código mais legível. Hospedei em núvem utilizando a Versel.
